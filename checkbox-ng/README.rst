Checkbox-ng
===========

``Checkbox-ng`` is a hardware testing tool useful for certifying laptops,
desktops, servers and IOT devices with Ubuntu.

Installation
============

$ sudo add-apt-repository ppa:checkbox-dev/edge
$ sudo apt-get update
$ sudo apt-get install checkbox-ng
